## cffi與單元測試

+ [程式範例](https://github.com/WaveGIS-Co/data_analyze_c_v/tree/master/anomaly_cleaner_v1)

```python
from cffi import FFI

ffi = FFI()

# 宣告你要使用的 C 函數接口
ffi.cdef("""
    int add(int a, int b);
""")

# 指定要加載的靜態庫或共享庫
# Linux/macOS 通常用 .so
# Windows 用 .dll
lib = ffi.dlopen("./libmylib.so")  # 或 Windows: "mylib.dll"

# 呼叫 C 函數
result = lib.add(5, 7)
print("5 + 7 =", result)
```
